# python_excel_data_cleaner
This Python script automatically cleans messy CSV files and generates Excel reports.

##Features
1.Removes missing values 
2.Removes negative expenses
3.Standardizes item names
4.Calculates category total
5.Exports for a matted Excel report

##Technologies used
1.Python
2.pandas

##How to run
1.Place CSV files inside the input_files folder
2.run excel_data_cleaner.py
3.clean Excel reports will appear in output_files
